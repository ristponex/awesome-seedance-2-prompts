## 关于 Seedance 2.0

**Seedance 2.0** 是字节跳动于 **2026 年 2 月** 发布的新一代 AI 视频生成模型。它是首个在生产环境中实现 **统一多模态视听联合生成** 的模型 —— 通过单一架构从一个提示词生成电影级视频、同步音频、对白、音乐和多镜头叙事内容。

### 核心能力

| 能力 | 详情 |
|:-----|:-----|
| **统一视听生成** | 单模型一次生成视频 + 同步音频 + 对白 |
| **复杂运镜** | 推拉变焦、焦点切换、跟踪拍摄、主观视角切换、希区柯克变焦等 |
| **多镜头叙事** | 生成保持角色和场景一致性的连贯多镜头序列 |
| **8+ 语言唇形同步** | 支持中文、英文、日文、韩文等 8 种以上语言的音素级唇形同步 |
| **角色一致性** | 通过图像/视频参考（`@image1`、`@video1`）保持跨镜头角色一致性 |
| **电影级画质** | 好莱坞级别的视觉质量，包含逼真的光照、景深和运动模糊 |

### 关键参数

| 参数 | 规格 |
|:-----|:-----|
| 最高分辨率 | 最高 1080p |
| 视频时长 | 5秒 – 20秒/次生成 |
| 宽高比 | 16:9、9:16、1:1、4:3、3:4 |
| 音频通道 | 立体声 |
| 帧率 | 24fps / 30fps |
| 输入模式 | 文生视频、图生视频、视频生视频 |
| 参考支持 | 多图参考（`@image1`、`@image2`）、视频参考（`@video1`） |

### 应用场景

- **影视制作与预可视化** — 快速故事板转视频、概念可视化、特效预览
- **广告与营销** — 产品广告、品牌推广、社交媒体广告


### Python 示例

```python
import requests
import time

# Atlas Cloud API 配置
API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.atlascloud.ai/api/v1/model"

def generate_video(prompt: str, duration: int = 10, aspect_ratio: str = "16:9"):
    """
    调用 Seedance 2.0 模型生成视频

    参数:
        prompt: 视频生成提示词
        duration: 视频时长（秒）
        aspect_ratio: 视频宽高比
    返回:
        视频生成任务的结果
    """
    # 提交视频生成请求
    response = requests.post(
        f"{BASE_URL}/generateVideo",
        headers={
            "Content-Type": "application/json",
            "Authorization": f"Bearer {API_KEY}"
        },
        json={
            "model": "seedance-v1.5-pro",
            "input": {
                "prompt": prompt,
                "negative_prompt": "模糊, 低画质, 变形",
                "duration": duration,
                "aspect_ratio": aspect_ratio
            }
        }
    )

    result = response.json()
    task_id = result.get("task_id")
    print(f"任务已提交，ID: {task_id}")

    # 轮询检查生成状态
    while True:
        status_resp = requests.get(
            f"{BASE_URL}/task/{task_id}",
            headers={"Authorization": f"Bearer {API_KEY}"}
        )
        status = status_resp.json()

        if status["status"] == "completed":
            print("视频生成完成！")
            return status["output"]["video_url"]
        elif status["status"] == "failed":
            raise Exception(f"生成失败: {status.get('error')}")

        time.sleep(5)  # 每5秒检查一次状态

# 使用示例
video_url = generate_video(
    prompt="镜头缓慢后拉（展现完整街景）并跟随女主角移动。风吹起她裙摆，"
           "她走在19世纪伦敦的街道上。煤气灯在鹅卵石街道上闪烁。"
           "电影级光照，胶片颗粒感，变形宽荧幕镜头眩光。",
    duration=15,
    aspect_ratio="16:9"
)
print(f"视频下载链接: {video_url}")
```

<summary><b>3.1 电梯希区柯克变焦</b> — 经典眩晕/推拉变焦效果</summary>

**提示词：**

### 运镜关键词

| 关键词 | 效果 | 最适用场景 |
|:-------|:-----|:-----------|
| `固定机位` / `Fixed camera` | 静态镜头，无移动 | 对话场景、产品拍摄 |
| `镜头跟踪` / `Tracking shot` | 镜头跟随被摄体横移 | 行走场景、追逐戏 |
| `镜头推进` / `Push-in` | 向前推轨运动 | 营造紧张感、揭示 |
| `镜头后拉` / `Pullback` | 向后拉轨运动 | 场景揭示、全景建立 |
| `推拉变焦` / `Dolly zoom` | 变焦+反向推轨 | 心理紧张感 |
| `环绕拍摄` / `Orbital shot` | 围绕被摄体360度旋转 | 产品展示、英雄镜头 |
| `摇臂镜头` / `Crane shot` | 垂直升降运动 | 宏大揭示、开场 |
| `甩拍` / `Whip pan` | 快速水平横摇 | 转场、动作场景 |
| `变焦追焦` / `Rack focus` | 焦点在不同平面间切换 | 引导注意力、转场 |
| `主观镜头` / `POV shot` | 被摄者视角 | 沉浸式体验 |
| `航拍` / `Aerial shot` | 高空俯瞰 | 风景、全景建立 |
| `低角度` / `Low angle` | 镜头低于视平线 | 使被摄者显得强大 |
| `荷兰角` / `Dutch angle` | 倾斜画框 | 紧张感、不安定感 |

## 常见问题

<details>
<summary><b>什么是 Seedance 2.0？</b></summary>

Seedance 2.0 是字节跳动于 2026 年 2 月发布的最先进 AI 视频生成模型。它是首个实现统一多模态视听联合生成的生产级模型 — 单一模型从文本提示词生成电影级视频、同步音频、对白、音乐和多镜头叙事内容。CNN 将其描述为"中国最新的 AI 强大到让好莱坞恐慌。"

</details>

<details>
<summary><b>Seedance 2.0 与其他 AI 视频模型有何不同？</b></summary>

Seedance 2.0 的核心差异化在于其**统一多模态生成**。与竞争对手分别生成视频和音频不同，Seedance 2.0 在单一模型通道中同时生成两者，确保完美同步。它还支持复杂运镜（推拉变焦、跟踪、主观视角切换）、8+ 语言音素级唇形同步和多镜头叙事一致性 — 这些是大多数竞争对手缺少的能力。

</details>

<details>
<summary><b>运行 Seedance 2.0 最便宜的方式是什么？</b></summary>

最具性价比的方式是通过 [Atlas Cloud](https://www.atlascloud.ai?ref=JPM683)，每视频 **$0.222**（目前有 90% 折扣）。新用户首次充值还可获得 **25% 额外赠送**（最高 $100）。这比通常收费 $0.30 或更高的其他 API 提供商便宜很多。

</details>

<details>
<summary><b>Seedance 2.0 能生成音频和对白吗？</b></summary>

可以。Seedance 2.0 具备统一视听生成功能。您可以在提示词中直接指定对白（支持 8+ 语言唇形同步）、背景音乐、音效和环境音。模型作为同一生成过程的一部分生成同步音频 — 而非单独的后处理步骤。

</details>

<details>
<summary><b>Seedance 2.0 支持哪些分辨率和时长？</b></summary>

Seedance 2.0 支持最高 1080p 分辨率，多种宽高比（16:9、9:16、1:1、4:3、3:4）。视频时长范围为每次生成 5 到 20 秒。支持 24fps 和 30fps 帧率。

</details>

<details>
<summary><b>如何在多个视频中保持角色一致性？</b></summary>

使用参考系统：上传角色图像并在提示词中用 `@image1` 引用。例如：`@image1 参考图中的人走进一家咖啡馆...` 这将保留面部特征、身体比例和关键视觉特征。您也可以使用 `@video1` 来引用之前生成的动作和节奏。

</details>

<details>
<summary><b>Seedance 2.0 支持哪些语言的唇形同步？</b></summary>

Seedance 2.0 支持 8+ 语言的音素级唇形同步，包括英语、中文（普通话）、日语、韩语、法语、西班牙语、德语等。只需在提示词中用目标语言编写对白，模型将生成匹配的唇部动作。

</details>

<details>
<summary><b>可以将 Seedance 2.0 用于商业项目吗？</b></summary>

可以。通过 Atlas Cloud 等平台访问时，生成的内容可用于商业目的，包括广告、社交媒体营销、产品视频等。请始终查看您使用平台的具体服务条款。

</details>
